Para crear el ejecutable de mouseCam se debe realizar los siguientes pasos,

1- Compilar la biblioteca PUI sin habilitar glut. Una forma de hacer esto es ir al directorio ./PUI y utilizar el Makefile con este mismo nombre SIN definir el parametro GLUT_HABILITADO.

2- Compilar mouseCam con el Makefile que se encuentra dentro de este mismo directorio.
